🌙 DARK MODE
═════════════════════════════════════════════════════════════════════════════

Sistema de dark mode:
   • Se controla desde ThemeToggle.tsx
   • Agrega clase .dark al elemento <html>
   • Variables CSS se adaptan automáticamente
   • Se guarda en localStorage

Variable de ejemplo:
   :root { --bg-light: #ffffff; }
   .dark { --bg-light: #1f242d; }

🚀 PASOS SIGUIENTES
═════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAR DEPENDENCIAS:
    npm install

2️⃣  (Si tienes problemas con PowerShell):
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

3️⃣  EJECUTAR EN DESARROLLO:
    npm run dev

4️⃣  ABRIR EN NAVEGADOR:
    http://localhost:3000

5️⃣  EDITAR ESTILOS:
    Abre: src/styles/main.css
    Guarda y verás cambios automáticamente

6️⃣  PERSONALIZAR CONTENIDO:
    • Proyectos: src/data/projects.ts
    • Habilidades: src/data/skills.ts
    • Componentes: src/components/

7️⃣  BUILD PARA PRODUCCIÓN:
    npm run build && npm start
